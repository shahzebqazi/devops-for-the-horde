Mission

What this is for

  • Bootstrap a Mac fast — paths from fresh install → productive (tooling, defaults, verification).
  • Personal domains — computing, game dev, pro audio & music, coding, 3D, AI lab (expand as you add repos).
  • Solo-first, fork-friendly — optimize for your machine; others can borrow ideas and open issues.

Workflow

Horde ops

  1. config/master.yaml — single source of truth for what belongs in the horde.
  2. Satellite repos — own app-specific automation; this repo links and summarizes.
  3. Cursor + agents — AGENTS.md and .cursor/rules for repeatable README/site sync (work work).
